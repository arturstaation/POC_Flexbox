O que é FlexBox

Durante muito tempo, as únicas tecnicas disponíveis para a criação de layouts usando CSS e posicionamento de elementos, e com boa compatibilidade entre browsers, eram float e position. Além disso, manipulações entre elementos pai-filho era uma outra tarefa complexa. Visto isso, essa ferramente foi criada para simplificar, sem perder a funcionalidade, essas tarefas, além de aumentar a eficiencia.

Que tal afrondar mais um pouco essa definição?

Flexbox, ou Flexible Box Layout, é um modelo de layout em CSS que facilita a criação de layouts complexos e responsivos. Ele foi desenvolvido para resolver problemas comuns com o posicionamento e alinhamento de elementos dentro de um contêiner. Alem disso, uma das principais ideias por trás desse recurso é possuir a habilidade de alterar o tamanho dos elementos de forma que preencham da melhor maneira o container onde estão inseridos.

Princiapais Conceitos

Para entender e manipular melhor o layout flexivel, você precisa saber alguns conceitos.

Start

Start é considerado o inicio do container flexivel, sendo o ponto mais a esquerda, tratando do eixo horizontal, ou mais acima, tratando do eixo vertical.

End

End é considerado o final do container flexivel, sendo o ponto mais a direita, tratando do eixo horizontal, ou mais abaixo, tratando do eixo vertical.

Eixo principal

O eixo principal, ou primario, é aquele que define a direção onde serão inseridos items.

Eixo transversal

O eixo transversal, ou perpendicular, recebe esse nome devido a sua orientação: corta o eixo principal.

Elemento Pai

O elemento pai é um elemento que contém um ou mais elementos filhos.Esses elementos filhos estão aninhados dentro do elemento pai e são, portanto, descendentes diretos ou indiretos desse elemento

Elemento Filho

O elemento filho é um elemento que está diretamente contido dentro de outro elemento, que é o seu elemento pai.

Como usar Flexbox

Agora que você já entende os conceitos básicos , vamos à prática. Para começar, adicione a seguinte propriedade ao seletor CSS do seu elemento: display: flex.

Mais importante do que saber como aplicar essa propriedade é entender quando e onde usá-la. A propriedade Flexbox deve ser aplicada a um elemento pai para organizar o layout dos seus elementos filhos. Isso permite que você controle a distribuição do espaço e o alinhamento dos itens dentro do contêiner.
